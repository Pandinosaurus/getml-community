Example 6: Units (part 2)
--------------------------

For a full working example, please refer to :ref:`example_06_same_units_2.R` and :ref:`example_06_same_units_3.R`.

The idea of units also applies to discrete and categorical variables. Don't worry - AutoSQL would never compare categorical to numerical, categorical to discrete or discrete to numerical variables, even if you decide to assign them the same unit for some reason.

Suppose our problem is like this:

.. code-block:: sql

    SELECT t1.*,
    COALESCE( COUNT( * ), 0 ) AS target
    FROM POPULATION_TABLE t1
    LEFT JOIN PERIPHERAL_TABLE t2
    ON t1.join_key = t2.join_key
    WHERE (
       ( t1.column_01 = t2.column_01 )
    ) AND t2.time_stamps <= t1.time_stamps
    GROUP BY t2.join_key;

If we do not include the information that *column_01* in POPULATION_TABLE and PERIPHERAL_TABLE share the same unit, AutoSQL would have to create a special case for each possible manifestation of that variable. It is not impossible, but it is inefficient and the resulting SQL code would not be very beautiful.

High-level API
^^^^^^^^^^^^^^^^^^

The way units are assigned works exactly as for the previous example:

.. code-block:: r

    units = list()

    units[["column_01"]] = "unit_01"

And fitting is very similar as well, with the exception that we need to declare both *column_01* as categorical variables:

.. code-block:: r

    population.placeholder <- create.autosql.placeholder(
        "POPULATION",
        categorical=c( "column_01" ),
        join.keys=c( "join_key" ),
        time.stamps=c( "time_stamp" ),
        targets=c( "targets" )
    )

    peripheral.placeholder <- create.autosql.placeholder(
        "PERIPHERAL",
        categorical=c( "column_01" ),
        join.keys=c( "join_key" ),
        time.stamps=c( "time_stamp" )
    )

    population.placeholder <- join.autosql.placeholder(
        population.placeholder,
        peripheral.placeholder,
        "join_key",
        "time_stamp"
    )

    model = create.autosql.model(
        name="Model 1",
        population=population.placeholder,
        peripheral=list(peripheral.placeholder),
        aggregation=list(        
        autosql.count, 
        autosql.sum,
        autosql.min
        ),
        units=units, # passing the units to the model
        use.timestamps=TRUE,
        num.features=10,
        max.length=2,
        fast.training=FALSE,
        min.num.samples=200,
        shrinkage=0.0,
        grid.factor=1.0,
        share.aggregations=1.0,
        num.threads=6,
        regularization=0.01,
        predictor=autosql.create.xgboost.regressor()
    )

    send.autosql.model(model)

    model = fit.autosql.model(
        model,
        population.table,
        peripheral.table
    )

Low-level API
^^^^^^^^^^^^^^^^^^

Here is how setting units works for the low-level API:

.. code-block:: r

    #----------------
    # Set up units

    units = list()

    units[["column_01"]] = "unit_01"

    # -------------------------------------------------
    # Load data into engine

    population.table.on.engine <- create.autosql.data.frame(
        name="POPULATION",
        categorical=c( "column_01" ),
        join.keys=c( "join_key" ),
        time.stamps=c( "time_stamp" ),
        targets=c( "targets" ),
        units=units # Passing units to the data frame
    )

    population.table.on.engine <- send.autosql.data.frame(
        population.table.on.engine,
        population.table
    )

    peripheral.table.on.engine <- create.autosql.data.frame(
        name="PERIPHERAL",
        categorical=c( "column_01" ),
        join.keys=c( "join_key" ),
        time.stamps=c( "time_stamp" ),
        units=units # Passing units to the data frame
    )

    peripheral.table.on.engine <- send.autosql.data.frame(
        peripheral.table.on.engine,
        peripheral.table
    )

Since you have already set the units in the data frame, you do not have to pass it to the model anymore:

.. code-block:: r

    population.placeholder <- create.autosql.placeholder("POPULATION")

    peripheral.placeholder <- create.autosql.placeholder("PERIPHERAL")

    population.placeholder <- join.autosql.placeholder(
        population.placeholder,
        peripheral.placeholder,
        "join_key",
        "time_stamp"
    )

    model = create.autosql.model(
        population=population.placeholder,
        peripheral=list(peripheral.placeholder),
        aggregation=list(        
        autosql.count, 
        autosql.sum,
        autosql.min
        ),
        use.timestamps=TRUE,
        num.features=10,
        max.length=2,
        fast.training=FALSE,
        min.num.samples=200,
        shrinkage=0.0,
        grid.factor=1.0,
        share.aggregations=1.0,
        num.threads=6,

    )

    send.autosql.model(model)
